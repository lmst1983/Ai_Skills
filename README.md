# AI Skills

Robin 个人维护的 AI、Codex 和 ChatGPT Skill 集合。

## 仓库说明

本仓库用于保存、维护和版本管理个人开发的 AI Skills。

每个 Skill 使用独立目录管理，并至少包含一个 `SKILL.md` 文件。

## 目录结构

```text
Ai_Skills/
├── README.md
├── .gitignore
└── skills/
    └── codex-chat-history-cleaner/
        ├── SKILL.md
        ├── agents/
        ├── references/
        └── scripts/

```
